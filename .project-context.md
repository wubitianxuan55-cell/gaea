# Project Context: Gaea Neural OS (Frontend-Heavy)

## 🎯 Current Status
The project is a high-fidelity web application built with React, Vite, and Tailwind CSS. It features a triple-mode interface:
1. **Web Mode**: Landing pages with intro pages for Desktop/Mobile.
2. **Desktop OS Mode**: A functional virtual OS environment with windows, taskbar, and theme engine.
3. **Mobile Platform Mode**: A specialized mobile neural interface UI.

## 🛠 Tech Stack
- **Framework**: React 11 (Vite)
- **Animation**: `motion` (framer-motion)
- **Icons**: `lucide-react`
- **Styling**: Tailwind CSS (Native Support)
- **State Management**: React Context & Hooks

## 📂 Key Architecture
- `/src/platforms/web/`: Main web landing and entry logic.
- `/src/platforms/desktop/`: Desktop-specific intro and node logic.
- `/src/platforms/mobile/`: Mobile UI and intro logic.
- `/src/components/DesktopUI.tsx`: The core logic for the Virtual OS (Windows, Spotlight, Kernel Monitor).
- `/src/components/LandingSections.tsx`: Web homepage modular sections.

## 🎨 Theme Specs
Standardized themes (Celestial, Nebula, Cyber) are managed via `theme` state in `DesktopUI.tsx`.
